De-Annotate/Unlink
===================

.. |C| replace:: **Check**

To test in: 

Web, Insight


Purpose of the test is to check the De-annotate (alias Unlink, alias Remove) functionality This is another functionality than Delete (which is covered by a different scenario).

#. Go to right-hand pane in Insight

#. Unlink tags

   - Unlink particular tags (left click - chose Unlink in Insight [:download:`001 </images/testing_scenarios/DeAnnotateUnlink/001.png>`, :download:`002b </images/testing_scenarios/DeAnnotateUnlink/002b.png>`], click on minus in Web to Remove)
   - Unlink all tags added by you/by others (minus sign at Tags in Insight only) [:download:`002 </images/testing_scenarios/DeAnnotateUnlink/002.png>`]

#. Repeat step 2 on attachments (Web: choose minus sign to remove - not cross to delete) [:download:`003 </images/testing_scenarios/DeAnnotateUnlink/003.png>`, :download:`004 </images/testing_scenarios/DeAnnotateUnlink/004.png>`]

#. Remove image from Dataset (Web only - icon in left hand pane)

   - |C| Check the image appears in the orphaned images in case it is not linked to other datasets.
