View
=====



**To test in**: Web, Insight

**Purpose of the test** is to test the View feature on multiple z/t image and to check the z-Projection creation functionality

#. Import an image with several channels/timepoint/z-sections. [:download:`001 </images/testing_scenarios/View/001.png>`]

#. Turn on/off channels, use z-slider and t-slider.3. Play channels movie and z/t movie.

#. Go to the split-view (=multi-channel in Web). Turn channels on/off, user z and t-sliders.

#. Toggle back between views.

#. Turn on/off color/greyscale

#. Modify the rendering settings.

#. Projection tabs [a tab in the Full Viewer next to Image tab]:[:download:`002 </images/testing_scenarios/View/002.png>`] (Insight only) - in Web, click on "Max Intensity" instead

   - Preview the projection [:download:`002 </images/testing_scenarios/View/002.png>`]
   - Change the projection range.
   - Change the projection type [Maximum or Mean - double arrow menu next to "Intensity" [:download:`003 </images/testing_scenarios/View/003.png>`]].
   - Project the image.[:download:`004 </images/testing_scenarios/View/004.png>`, :download:`005 </images/testing_scenarios/View/005.png>`]
   - |C| View the result [:download:`006 </images/testing_scenarios/View/006.png>`]

#. Save as (Insight only):

   - JPEG, PNG - use the icon in the projection window, see [:download:`007 </images/testing_scenarios/View/007.png>`, :download:`008 </images/testing_scenarios/View/008.png>`]
   - Save split channels.[:download:`008 </images/testing_scenarios/View/008.png>`]
   - Save with all channels turned off.
   - |C| the saved files.
