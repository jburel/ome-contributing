Render
=======



**To test in**: Web, Insight

**Purpose of the test** is to make sure rendering is smooth and changes are applied consistently to all visualisations of the images and all the options of rendering are functional

#. Copy settings from image [right-click on the thumbnail in central pane].[:ref:`RenderScreen001`] (Insight only)

#. Select a collection of images and paste settings.[:ref:`RenderScreen002`] (Insight only)

   - |C| Make sure that the thumbnails in the main pane are updated.

#. Reset default settings.[:ref:`RenderScreen003`] (Web - in Full Viewer only, for one image)

   - |C| Make sure thumbnails main viewer are updated

#. Set min/max settings. (Insight only)

   - |C| Make sure thumbnails main viewer are updated

#. Set owner's settings. (Insight only)

   - |C| Make sure thumbnails main viewer are updated [:ref:`RenderScreen004`]

#. If applicable (does not work for Big images):

   - Open an image belonging to another user.
   - Modify the settings in the "Preview" pane on the right.[:ref:`RenderScreen005`] Save your own using the "Apply" icon [Insight]. or Save button (Web) [:ref:`RenderScreen007`]
   - |C| Make sure thumbnails in central pane are updated

#. Repeat the steps 1 - 5. with Big Images (Find suitable images here [:download:`TestingSet </downloads/ImportTestingSetTable.pdf>`] or here [:download:`TestingTemplate </downloads/ImportTestingTemplate.pdf>`])
