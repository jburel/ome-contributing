Render
=======



**To test in**: Web, Insight

**Purpose of the test** is to make sure rendering is smooth and changes are applied consistently to all visualisations of the images and all the options of rendering are functional

#. Copy settings from image [right-click on the thumbnail in central pane].[:ref:`RenderScreen001`][:download:`001 </images/testing_scenarios/Render/001.png>`] (Insight only)

#. Select a collection of images and paste settings.[:ref:`RenderScreen002`][:download:`002 </images/testing_scenarios/Render/002.png>`] (Insight only)

   - |C| Make sure that the thumbnails in the main pane are updated.

#. Reset default settings.[:ref:`RenderScreen003`][:download:`003 </images/testing_scenarios/Render/003.png>`] (Web - in Full Viewer only, for one image)

   - |C| Make sure thumbnails main viewer are updated

#. Set min/max settings. (Insight only)

   - |C| Make sure thumbnails main viewer are updated

#. Set owner's settings. (Insight only)

   - |C| Make sure thumbnails main viewer are updated [:ref:`RenderScreen004`][:download:`004 </images/testing_scenarios/Render/004.png>`]

#. If applicable:

   - Open an image belonging to another user.
   - Modify the settings in the "Preview" pane on the right.[:ref:`RenderScreen005`][:download:`005 </images/testing_scenarios/Render/005.png>`] Save your own using the "Apply" icon [Insight].[:ref:`RenderScreen006`][:download:`006 </images/testing_scenarios/Render/006.png>`] or Save button (Web) [:ref:`RenderScreen007`][:download:`007 </images/testing_scenarios/Render/007.png>`].
   - |C| Make sure thumbnails in central pane are updated
