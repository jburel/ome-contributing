Create
======



**To test in**: Web, Insight

**Purpose of the test** is to check that the creation of new entities is smooth, valid and sensible and the appropriate menu commands work and correspond (right-click and main menu)

If you test work on others' data (permissions) go first to :ref:`Section 2: Work on others data <label-Create>`

**Section 1: Work on your own data**

#. Create new Project/Dataset/Screen* Create using right-click [:ref:`CreateScreen001`]
 
   - Create using the icon [:ref:`CreateScreen002`, :ref:`CreateScreen003`, :ref:`CreateScreen004`]
   - Create using the main menu (Insight only) [:ref:`CreateScreen005`]

#. Create new Tag Set/Tag (Menu:File>New, Insight only) [:ref:`CreateScreen005`]

#. Select an image. Create New dataset from Selection (right-click - Insight only)

   - |C| that a new dataset is created with the selected images inside

#. Right-click on dataset, choose "Newâ€¦."

   - |C| that the "New project" window appears

**Section 2: Work on other users' data**

#. Switch to Group View in Insight, put "All members" to display in Web.

#. Do step 1 in Section 1 only creating Datasets under others' Projects (it is not possible to create Project or Screen under the name of another user)

#. Do step 3 in Section 1 working on other users' images.

#. Switch to User View in Insight, put a particular user to display in Web and do steps 2 and 3 in Section 2.
