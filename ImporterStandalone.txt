Importer Standalone
===================



**To test in**: Standalone Importer UI

**Purpose of the test** is to check the Importer Standalone functionality in importing images/screens
 
   - |C| that the application start and you can connect to the server [:ref:`ImporterStAScreen001`][:ref:`ImporterStAScreen002`][:download:`001 </images/testing_scenarios/ImporterStandalone/001.png>`, :download:`002 </images/testing_scenarios/ImporterStandalone/002.png>`]

#. Import data in various groups if applicable [:ref:`ImporterStAScreen003`][:ref:`ImporterStAScreen004`][:ref:`ImporterStAScreen005`][:ref:`ImporterStAScreen006`][:download:`003 </images/testing_scenarios/ImporterStandalone/003.png>`, :download:`004 </images/testing_scenarios/ImporterStandalone/004.png>`, :download:`005 </images/testing_scenarios/ImporterStandalone/005.png>`, :download:`006 </images/testing_scenarios/ImporterStandalone/006.png>`]

#. Create New Project/Dataset to import the images into.[:ref:`ImporterStAScreen007`][:download:`007 </images/testing_scenarios/ImporterStandalone/007.png>`] Toggle the icon to Screen in the location selection window.[:ref:`ImporterStAScreen008`][:download:`008 </images/testing_scenarios/ImporterStandalone/008.png>`] Create New Screen to import the plates into.
