Scripts Client
===============



**To test in**: Web, Insight


**Purpose of the test** is to test the scripting functionalities, mainly checking the script launching on various targets and reporting about the script run

#. Select a dataset

#. Open list of available scripts.[:ref:`ScriptClientScreen001`][:download:`001 </images/testing_scenarios/ScriptsClient/001.png>`]

#. Select a script running on datasets.[:ref:`ScriptClientScreen002`][:ref:`ScriptClientScreen003`][:download:`002 </images/testing_scenarios/ScriptsClient/002.png>`, :download:`003 </images/testing_scenarios/ScriptsClient/003.png>`]

   - |C| that the data type is set to dataset and the id of the dataset is displayed in the id field.

   - Select other data type if applicable,

   - |C| that the id is removed.[:ref:`ScriptClientScreen002`][:ref:`ScriptClientScreen003`][:download:`002 </images/testing_scenarios/ScriptsClient/002.png>`, :download:`003 </images/testing_scenarios/ScriptsClient/003.png>`]

#. Select the initial data type and enter the initial id.

#. Run the script and

   - |C| that an entry is added to the activities section [:ref:`ScriptClientScreen004`][:ref:`ScriptClientScreen005`][:download:`004 </images/testing_scenarios/ScriptsClient/004.png>`, :download:`005 </images/testing_scenarios/ScriptsClient/005.png>`]

#. Click on the result link e.g.

   - |C| View if the script ran to completion w/o errors.[:ref:`ScriptClientScreen006`][:ref:`ScriptClientScreen007`][:download:`006 </images/testing_scenarios/ScriptsClient/006.png>`, :download:`007 </images/testing_scenarios/ScriptsClient/007.png>`]

#. Repeat all steps with another type e.g. Image
