Metadata Display
=================



**To test in**: Web, Insight

**Purpose of the test** is to test the metadata display, and check they display consistently in Web and Insight

#. Import an image rich in metadata for example 'data_repo/test_images_metada/leica-lif/Rosier.lif.2' or see the set of images here [:download:`here <examples/ImportTestingSetTable.pdf>`] Compare the display between web and insight i.e. Make sure the same values are displayed [:ref:`MetadataScreen001`][:ref:`MetadataScreen002`][:download:`001 </images/testing_scenarios/MetadataDisplay/001.png>`, :download:`002 </images/testing_scenarios/MetadataDisplay/002.png>`]

   - |C| the units

#. Show or hide unset fields [not populated fields - click on "show unset fields" in the metadata tabs].[:ref:`MetadataScreen003`][:ref:`MetadataScreen004`][:download:`003 </images/testing_scenarios/MetadataDisplay/003.png>`, :download:`004 </images/testing_scenarios/MetadataDisplay/004.png>`]

   - |C| Make sure they are the same in both clients (Web and Insight)
   - |C| that the user cannot edit the fields.
