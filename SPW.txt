SPW
====

.. |C| replace:: **Check**

To test in: 

Web, Insight


Purpose of the test is to check a plate/screen (with multiple Fields per well) import and viewing



1. In Importer, in the Location dialog. Switch from Project/Dataset to Screen. Import a plate with multiple fields for example ome/data_repo/test_images_good/cellworx/hms/plate1/plate1_802.pnl

 - |C| Refresh Datamanager. Check that a Plate Run (Plate Acquisition) is created in "Screens" tab [:download:`001 </images/testing_scenarios/SPW/001.png>`] (Insight only)

2. Double-click on Plate or Plate Run.

 - |C| Both actions should display in the central panel.[:download:`002 </images/testing_scenarios/SPW/002.png>`, :download:`003 </images/testing_scenarios/SPW/003.png>`]

3. Double-click on a well/field to open viewer [:download:`004 </images/testing_scenarios/SPW/004.png>`, :download:`005 </images/testing_scenarios/SPW/005.png>`]

4. Select a new Field in the dropdown menu above the central panel [:download:`006 </images/testing_scenarios/SPW/006.png>`]

 - |C| All thumbnails should be updated.

5. Toggle on and off the icon allowing you to preview on mouse over above the central panel.[:download:`007 </images/testing_scenarios/SPW/007.png>`]

6. Open the view of all Fields - icon to the left of field choice menu, above central panel.[:download:`008 </images/testing_scenarios/SPW/008.png>`] (Insight only)

 - Select "Spatial" view [:download:`009 </images/testing_scenarios/SPW/009.png>`, :download:`010 </images/testing_scenarios/SPW/010.png>`]
 - |C| Check that the spatial relation of the fields is displayed properly.
 - Select "As a row" view [:download:`011 </images/testing_scenarios/SPW/011.png>`]
