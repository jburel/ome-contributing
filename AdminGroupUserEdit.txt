Admin: Group/User Edit
========================



**To test in**: Web, Insight

**Purpose of the test** is to check the Editing and changing of Group/User as administrator

#. Edit group [:ref:`AdminGrUsEdScreen001`][:ref:`AdminGrUsEdScreen002`][:ref:`AdminGrUsEdScreen003`]

#. Change name/description/permissions and Save

   - |C| the parameters have been changed correctly 

#. Add and remove users (=members) and owners 

   - using Edit group window (Web only) [:ref:`AdminGrUsEdScreen003`]
   - using "Add existing users" from right-click menu (Insight only) [:ref:`AdminGrUsEdScreen004`]
   - using Copy/Cut and Paste from main menu (Insight only)
   - using Copy/Cut and Paste via icons & right-click (Insight only)
   - using Drag & Drop (Insight only)

   - |C| the users have been added/removed to/from the groups
 
#. Edit user [:ref:`AdminGrUsEdScreen005`][:ref:`AdminGrUsEdScreen006`][:ref:`AdminGrUsEdScreen007`]

   - change Login Name/Password/Status (Insight)
   - change Names/e-mail/Institution/Status (Web - at present no Group Owner option)
   - Save changes
   - |C| the changes have been made appropriately
