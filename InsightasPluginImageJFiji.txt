Insight as Plugin ImageJ/Fiji
==============================



**To test in**: ImageJ, Fiji

**Purpose of the test** is to check that the OMERO insight works as plugin for ImageJ/ImageJ2/Fiji and the loci_tools.jar missing is not causing a crash.

**Test on Linux as well as Windows (XP and Win7 if possible)**

#. Download the OMERO.insight-ijâ€¦ build and unzip it [or copy it into] in imageJ/plugins folder [:ref:`InsightIJI001`]

#. Download the latest stable version of ImageJ for Windows/Mac/Linux as appropriate from `ImageJ homepage <http://rsbweb.nih.gov/ij/download.html>`_.

#. Do not add "loci_tools.jar" to the ImageJ/plugins folder. If it is there, remove it.

#. Start imageJ (resp. Fiji) and go Plugins/Connect to OMERO

   - |C| Dialog should indicate that "loci_tools.jar" is missing.[:ref:`InsightIJI002`]

#. Click on the hyperlink in the dialog. Download loci_tools.jar.

#. Close ImageJ. Copy loci_tools.jar into ImageJ/Plugins folder.

#. Repeat step 3, this time

   - |C| Insight splashscreen should be displayed.

#. Log in.

#. Go to the left-hand pane and select an image or import one if no images available.

#. Right-click on the image,

   - |C| The "View" menu item should have the "View in ImageJ" option [:ref:`InsightIJI003`][:download:`003 </images/testing_scenarios/InsightasPluginImageJFiji/003.png>`]
   - Select the "View in imageJ" option.
   - |C| The image opens in imageJ

#. Double-click on the image.

   - |C| The image opens in imageJ.

#. Go through all steps using ImageJ64 and the platform independent version of ImageJ (download from `ImageJ homepage <http://rsbweb.nih.gov/ij/download.html>`_).

#. Download the latest stable version of Fiji from `Fiji homepage <http://fiji.sc/Fiji>`_. 

#. (FS-only) Note that you have to update Fiji in order to run Bio-Formats 5 with your Fiji. Go to the `Fiji update page <http://fiji.sc/Bio-Formats#Daily_builds>`_). Follow the instructions there.

#. Download the Insight IJ/Fiji plugin, put it into the Plugin folder of Fiji (expose Fiji's plugin folder on Mac via right-click and Show package content).

#. Restart Fiji and go Plugins > OMERO.

   - |C| that Insight starts and you can connect and open an image.
