Insight as Plugin ImageJ/Fiji
==============================



**To test in**: ImageJ

**Purpose of the test** is to check that the OMERO insight works as plugin for ImageJ/IJ64/Fiji and the loci_tools.jar missing is not causing a crash.

#. Download the OMERO.insight-ijâ€¦ build and unzip it [or copy it into] in imageJ/plugins folder [:ref:`InsightIJIScreen001`][:download:`001 </images/testing_scenarios/InsightasPluginImageJFiji/001.png>`].

#. Do not add "loci_tools.jar" to the imageJ/plugins. If it is there, remove it.

#. Start imageJ (resp. Fiji) and go Plugins/Connect to OMERO

   - |C| Dialog should indicate that "loci_tools.jar" is missing.[:ref:`InsightIJIScreen002`][:download:`002 </images/testing_scenarios/InsightasPluginImageJFiji/002.png>`].

#. Click on the hyperlink in the dialog. Download loci_tools.jar.

#. Close ImageJ. Copy loci_tools.jar into ImageJ/Plugins folder.

#. Repeat step 3, this time

   - |C| Insight splashscreen should be displayed.

#. Log in.

#. Go to the left-hand pane and select an image or import one if no images available.

#. Right-click on the image,

   - |C| The "View" menu item should have the "View in ImageJ" option [:ref:`InsightIJIScreen003`][:download:`003 </images/testing_scenarios/InsightasPluginImageJFiji/003.png>`]
   - Select the "View in imageJ" option.
   - |C| The image opens in imageJ

#. Double-click on the image.

   - |C| The image opens in imageJ.

#. Try with ImageJ and ImageJ64

#. Repeat using Fiji
