User/Group Manipulation
========================

.. |C| replace:: **Check**

To test in:

Web, Insight


Purpose of the test is to check the viewing/display of groups/users and their data

#. Add several groups to the display (Insight only) Switch between groups. #. Add several users to the display [:download:`004 </images/testing_scenarios/UserGroupManipulation/004.png>`, :download:`005 </images/testing_scenarios/UserGroupManipulation/005.png>`] (Insight only)

#. Switch between users.

#. Browse the user's data.

#. Remove users from display.(Insight only)

#. Remove groups from display.(Insight only)

#. Log off and log in as a new user.

   - |C| Check you can browse data
   - |C| Check you can view images.
