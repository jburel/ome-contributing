Editor from Insight
===================



**To test in**: Insight

**Purpose of the test** is to check the functionality (edit, save, re-open) of Editor as launched from Insight

#. Launch editor from the Data Manager toolbar [:ref:`EditorfInsight001`][:download:`001 </images/testing_scenarios/EditorfromInsight/001.png>`]. Close and launch the editor from the main menu.

#. Open a "New blank file", add few fields

#. Save locally and save to the server using main menu File/Save As...[:ref:`EditorfInsight002`][:ref:`EditorfInsight003`][:ref:`EditorfInsight004`][:ref:`EditorfInsight005`]

#. Go back to the Attachments tabs in the left-hand panel, find the [in step 3] created file and open it in the editor using double-click [:ref:`EditorfInsight006`]

   - Add more text to the file.
   - Save and close the editor.

#. Repeat step 4.
